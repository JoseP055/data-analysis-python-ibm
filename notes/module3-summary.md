# Módulo 3: Análisis Exploratorio de Datos (EDA)

## Descripción general
En este módulo cambié la perspectiva: pasé de *preparar* los datos a *entenderlos profundamente*.  
Aquí el objetivo fue claro: **descubrir patrones, relaciones y señales ocultas** que guían cualquier decisión analítica o modelo predictivo.

Durante este módulo aprendí a:

- Utilizar `pandas.describe()` para obtener una visión estadística instantánea: media, desviación estándar, cuartiles… todo lo necesario para entender el comportamiento general de las variables numéricas.  
- Explorar variables categóricas con `value_counts()`, esencial para ver distribuciones, categorías dominantes y posibles desequilibrios.  
- Visualizar distribuciones con **boxplots**, detectando rápidamente medianas, rangos y outliers.  
- Evaluar relaciones entre variables continuas con **scatter plots**, identificando patrones, tendencias y posibles correlaciones.  
- Aplicar `groupby()` para analizar cómo cambian métricas clave según distintos grupos o categorías.  
- Construir **tablas dinámicas** y **mapas de calor** para ver patrones complejos de forma visual y directa.

Finalmente entré al terreno crucial de la **correlación**:

- Aprendí el poder del *coeficiente de Pearson* para medir la fuerza y dirección de relaciones lineales.  
- Entendí el rol del **p-value** para confirmar si una correlación es significativa o solo ruido.  
- Practiqué regresiones simples usando `sns.regplot`, combinando estadística y visualización para interpretar relaciones con claridad.
